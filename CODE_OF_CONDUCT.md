# Code of Conduct

This project follows the
[Contributor Covenant Code of Conduct, v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be respectful, focus on what is best for the community, show
empathy. Harassment of any kind is not tolerated.

Reports should go to the maintainers via the email address in `SECURITY.md`.
